In Spring Boot REST APIs, you typically pass an access token in the **Authorization header** using the **Bearer** scheme.

### HTTP Request

```http
GET /employees
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI...
```

### Postman

1. Select **Authorization** tab.
2. Type: **Bearer Token**
3. Paste the access token.

Or add it manually in Headers:

```text
Key: Authorization
Value: Bearer <access_token>
```

### Using cURL

```bash
curl -X GET http://localhost:8080/employees \
-H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI..."
```

### Using RestTemplate

```java
HttpHeaders headers = new HttpHeaders();
headers.setBearerAuth(accessToken);

HttpEntity<String> entity = new HttpEntity<>(headers);

ResponseEntity<String> response =
        restTemplate.exchange(
                "http://localhost:8080/employees",
                HttpMethod.GET,
                entity,
                String.class);
```

### Using WebClient

```java
WebClient webClient = WebClient.builder()
        .baseUrl("http://localhost:8080")
        .build();

String response = webClient.get()
        .uri("/employees")
        .headers(h -> h.setBearerAuth(accessToken))
        .retrieve()
        .bodyToMono(String.class)
        .block();
```

### Read Token in Spring Boot Controller

```java
@GetMapping("/employees")
public ResponseEntity<String> getEmployees(
        @RequestHeader("Authorization") String token) {

    return ResponseEntity.ok(token);
}
```

### If Using Spring Security + JWT

The token is automatically extracted from:

```http
Authorization: Bearer <token>
```

and validated by the configured JWT filter. You can access the authenticated user as:

```java
Authentication authentication =
        SecurityContextHolder.getContext().getAuthentication();

String username = authentication.getName();
```

**Interview answer:** *Access tokens are usually passed in the HTTP Authorization header as `Bearer <token>`.*
