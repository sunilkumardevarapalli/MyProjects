Your approach is almost correct. There are a few syntax issues:

```java
String str = "aaabbcc";

Map<Character, Long> result =
        str.chars()
           .mapToObj(c -> (char) c)
           .collect(Collectors.groupingBy(
                   Function.identity(),
                   LinkedHashMap::new,
                   Collectors.counting()))
           .entrySet()
           .stream()
           .filter(e -> e.getValue() > 1)
           .collect(Collectors.toMap(
                   Map.Entry::getKey,
                   Map.Entry::getValue));
```

### Output

```java
{a=3, b=2, c=2}
```

### Simpler Version

```java
Map<Character, Long> counts =
        str.chars()
           .mapToObj(c -> (char) c)
           .collect(Collectors.groupingBy(
                   Function.identity(),
                   LinkedHashMap::new,
                   Collectors.counting()));

System.out.println(counts);
```

Output:

```java
{a=3, b=2, c=2}
```

### Interview-Friendly Alternative

Using Java 8 Streams:

```java
str.chars()
   .mapToObj(c -> (char) c)
   .collect(Collectors.groupingBy(
           Function.identity(),
           Collectors.counting()))
   .forEach((k, v) -> System.out.println(k + " -> " + v));
```

Output:

```text
a -> 3
b -> 2
c -> 2
```

The key correction is using method references:

```java
Map.Entry::getKey
Map.Entry::getValue
```

instead of

```java
Map.Entry.getKey
Map.Entry.getValue
```
