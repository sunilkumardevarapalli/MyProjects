If you want employee details while removing duplicate salaries, distinct() will not work because it removes duplicate Employee objects,
 not duplicate salary values.
 

Option 1: Keep the first employee for each salary (Recommended)

	List<Employee> uniqueEmployees = employees.stream()
			.collect(Collectors.toMap(
					Employee::getSalary,
					emp -> emp,
					(existing, duplicate) -> existing
			))
			.values()
			.stream()
			.sorted(Comparator.comparing(Employee::getSalary).reversed())
			.collect(Collectors.toList());
	 
Option 2: Get the 3rd highest salary employee details

	List<Employee> thirdHighestEmployees = employees.stream()
			.collect(Collectors.toMap(
					Employee::getSalary,
					emp -> emp,
					(e1, e2) -> e1
			))
			.values()
			.stream()
			.sorted(Comparator.comparing(Employee::getSalary).reversed())
			.skip(2)
			.limit(1)
			.collect(Collectors.toList());
 
Option 3: If you want all employees having the 3rd highest salary

	Double thirdHighestSalary = employees.stream()
        .map(Employee::getSalary)
        .distinct()
        .sorted(Comparator.reverseOrder())
        .skip(2)
        .findFirst()
        .orElse(null);

	List<Employee> result = employees.stream()
			.filter(emp -> emp.getSalary().equals(thirdHighestSalary))
			.collect(Collectors.toList());
